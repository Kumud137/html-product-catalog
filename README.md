# HTML Product Catalog

This is a simple product catalog website built using pure HTML.  
The project displays a list of products with images, descriptions, and prices.

## Purpose
This project is created for educational purposes to practice HTML structure, media elements, and basic web page layout.

## Technologies Used
- HTML5

## How to Use
1. Download or clone the repository.
2. Open the `index.html` file in any web browser.
3. Browse the product catalog.
